# Musify

A modern, open-source music streaming app built with Flutter. Musify allows you to stream music from various sources and manage your playlists seamlessly.

## Features

- Stream music from multiple sources
- Create and manage playlists
- Offline music playback
- Beautiful, intuitive user interface
- Multi-platform support (Android, iOS, Web, Desktop)
- Localization support for multiple languages
- Dark/Light theme support

## Installation

### Prerequisites

- Flutter SDK (>=3.35.7)
- Dart SDK (>=3.8.0 <4.0.0)

### Clone the Repository

```bash
git clone https://github.com/gokadzev/Musify.git
cd Musify
```

### Install Dependencies

```bash
flutter pub get
```

### Run the App

#### Android
```bash
flutter run
```

#### iOS
```bash
flutter run
```

#### Web
```bash
flutter run -d web
```

#### Desktop (Linux/Windows/macOS)
```bash
flutter run -d linux  # or windows/macos
```

## Building for Release

### Android APK
```bash
flutter build apk --release
```

### iOS
```bash
flutter build ios --release
```

### Web
```bash
flutter build web --release
```
